함수 실행을 안내할 때는 항상 해당 함수가 어느 파일(GAS 파일명)에 있는지 함께 명시한다.

예: "[Main.js](apps-script-new/Main.js)의 `setupTriggers` 실행해보세요."

**Why:** 사용자가 GAS 에디터에서 파일을 직접 찾아야 하므로 파일명이 없으면 불편함. 여러 번 지적했음 — 반드시 지켜야 함.

**How to apply:** 함수명을 언급할 때마다 예외 없이 파일명 병기. 단순 언급이든 실행 요청이든 동일하게 적용.
