번들 ID를 변경할 때는 빌드 전에 반드시 안내한다: "번들 ID를 변경하면 기기에 구 앱이 남습니다. 빌드 전에 아이폰에서 기존 앱을 먼저 삭제해주세요."

**Why:** Xcode는 새 번들 ID 앱을 설치하지만 구 번들 ID 앱은 자동 삭제하지 않음. 결과적으로 홈 화면에 동명의 앱이 두 개 생기고, 만료된 구 앱 아이콘을 탭해 "더 이상 사용할 수 없음" 에러가 반복됨 (2026-04-27 발생)

**How to apply:** 번들 ID 수정 커밋 전 또는 `PRODUCT_BUNDLE_IDENTIFIER` 변경 시 항상 이 안내를 먼저 출력
