description: 장중 손익 알림을 "워치 푸시"가 아니라 "텔레그램 푸시"라고 부른다
metadata:
  type: feedback
---

장중 손익 알림 기능(`tgPushPnL`)을 가리킬 때 **"텔레그램 푸시"**라고 부른다. "워치 푸시"라는 표현은 쓰지 않는다 (사용자 지시).

**Why:** 애플워치 한정으로 오해될 수 있고, 실제로는 텔레그램으로 가는 손익 푸시임.

**How to apply:** 대화·문서·커밋 메시지에서 "워치 푸시"·"워치 손익 알림" → "텔레그램 푸시"로 표기. 기존 docs의 "워치" 표현도 손대는 김에 같이 정리. 관련: [[project_market_report_pipeline]]
