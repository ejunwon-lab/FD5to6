작업 시작 전에 항상 방향과 방법을 먼저 설명하고, 사용자와 상의 후 승인을 받고 진행한다.

**Why:** 내 제약(Google Sheets 접근 불가 등)을 감안하지 않고 바로 실행해서 토큰을 낭비한 사례가 있었음. 사전에 상의했으면 불필요한 작업을 피할 수 있었음.

**How to apply:** 어떤 작업이든 "이렇게 하려고 합니다, 괜찮으신가요?" 먼저 물어보고 진행. 특히 외부 데이터나 접근 불가 리소스가 연관된 경우 반드시 확인.
