사용자는 *대시보드* 시트만 본다. *보유현황*, *종목상태*, *현재가_이력* 등은 모두 raw data용이고 직접 보지 않는다.

**Why:** 사용자가 명시적으로 "나는 대시보드만 볼 거야. 나머지는 대시보드를 완성하기 위한 raw data"라고 말함.

**How to apply:** 타임스탬프, 상태 정보, 수익률 등 사용자가 봐야 할 것은 무조건 대시보드에 표시. raw 시트에만 있으면 사용자가 볼 수 없음.
