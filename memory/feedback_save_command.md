`저장!` 라고 하면 세션 문서 저장 + memory 복사 + git add/commit/push를 한 번에 실행한다.
중간에 사용자에게 확인을 구하지 않고 끝까지 쭈욱 실행한다.

**Why:** "중간 저장해"와 "동기화해"를 매번 따로 입력하기 번거로워서 합친 단축 명령어. 확인 없이 자동 실행 원함.

**How to apply:** 사용자가 "저장!" 이라고 하면 두 작업을 순서대로 모두 멈추지 않고 실행.
