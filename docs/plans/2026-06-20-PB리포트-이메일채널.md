## 1. 외부 동작 가정 + 근거
- **MailApp.sendEmail은 `script.send_mail` 스코프 필요** → 현 appsscript.json에 없음 → 추가 필요. 근거: GAS 문서 + 현 oauthScopes에 mail 없음(직접 확인).
- **executeAs USER_DEPLOYING → MailApp이 소유자 Gmail로 발송, getEffectiveUser=소유자** → 근거: appsscript.json 직접 확인 + GAS 문서(웹앱 execute as me).
- **셀프발송**: 수신=발송=소유자 → 주소 하드코딩 0, `getEffectiveUser().getEmail()`. `userinfo.email` 스코프 이미 있음.
- **ANYONE_ANONYMOUS → GitHub이 action=emailReport POST 가능** → 근거: 텔레그램 pushPnL 입증.
- **운용수익률% 시계열 존재**: 추이기록 수익추이(U~AF, col21~32)에 일별 operatingRate. row5↓. → 근거: Trend.js:[금액] 헤더+코드.
- **market-report job guard가 이메일도 1회 보장**: "Skip if already sent"+found 조건을 이메일 스텝도 상속 → 지연 cron run은 guard로 skip(텔레그램과 동일, 6/19 멱등 실측). GAS에도 가벼운 dedup(같은 날·타입 1회) 추가로 이중.
- **텔레그램 4096 한도**: 현 KR ~[금액]자(4928바이트지만 한글 3바이트) → PB 격상 시 근접 → 프롬프트 가드(≤3500자) + send_telegram 분할 안전망. → 본문 실제 자 수는 `[검증 필요·dry_run]`.

## 2. 과거 부류 (errors.md)
- **🔴 oauthScopes 누락 → UrlFetchApp 권한 에러 + 사용자 재승인 (errors.md:141-145)** — *정확히 같은 부류.* MailApp 스코프 추가 = 새 권한 = 재승인. **회피책 그대로**: ① appsscript.json에 `script.send_mail` **명시** ② push_safe + 버전배포 ③ **사용자가 에디터에서 MailApp 호출 함수 1회 실행 → 권한 모달 승인**(이거 없으면 웹앱 MailApp 호출 시 권한 에러). 스코프는 프로젝트 단위 동의라 승인 후 모든 배포에서 작동.
- **멱등 순서 (errors.md:7-12, addTrade)** — 멱등 체크를 다른 게이트보다 먼저. 이메일 dedup도 발송 전 선점(tgPushPnL `tg_lastPushEpoch` 패턴 재사용).
- **환각 목표가 (errors.md, KR [금액]원)** — PB 격상해도 데이터 경계 유지: 밸류에이션 목표가 단정 금지, fetch한 값만. 출처 가드레일 강화.
- **원화 미노출 (privacy 게이트)** — recentReturns는 수익률%만(operatingRate), 원화 절대액 0. dry_run 본문 전수.
- **push|tail 트랩 (errors.md:40-45)** — 배포·push는 파이프 없이 exit code 직접.
- **best-effort 스케줄러** — 이메일은 market-report job 내 텔레그램 직후 발송(별도 스케줄 없음) → 무관.

## 3. 추론 가능 vs 실환경 전용
- **머리로 거름**: doPost action 분기(기존 패턴)·멱등(guard+dedup)·수익률% 변환(원화 0)·프롬프트 구조·분할 로직·발송 위치(GAS가 SMTP보다 인증 0 우세).
- **실환경에서만 (스모크/실측)**:
  - ① MailApp 스코프 추가 후 **재승인 동작** + 웹앱(execute as)에서 MailApp 권한 작동 → UrlFetchApp 전례로 거의 확실하나 **실행 전 미확정**. 사용자 1회 승인 후 emailReport POST → 도착으로만 확정.
  - ② `getEffectiveUser().getEmail()` 실제 주소 + Workspace 여부(쿼터) → GAS 실행으로 확인.
  - ③ PB 본문 실제 자 수(4096) → dry_run 생성물로.
  - ⚠️ 로컬·다른 IP 테스트로 "발송 됨" 판정 금지 — 실제 GitHub→GAS→Gmail 경로로만.
