# 2026-04-25 UI 개선

## 작업 내용

### 1. 기간별 수익 statsRow (ProfitHistoryView)
- 일평균에 `sub: " "` 추가 → 최고/최저 하루와 높이 정렬
- 최저 하루 색상: `worst.change < 0`일 때만 파란색 (기존: 항상 파란색)
- statCell 폰트 2단계 업: caption2→footnote, caption→subheadline

### 2. 대시보드 하단 4개 항목 폰트 (DashboardView)
- statItem: caption→subheadline, subheadline→callout, footnote→body
- fxItem: caption→subheadline, subheadline→callout

### 3. 참고지표 카테고리 테두리 (IndicatorsView)
- 카테고리별 컬러 테두리 추가 (lineWidth 2.5, cornerRadius 16)
- 섹션 타이틀 색상도 테두리 색과 통일
- 그림자도 같은 색 계열로 적용
- 색상: 한국시장(파랑), 한국선물(청록), 미국시장(빨강), 미국선물(주황), 상품(골드), 매크로(보라)
