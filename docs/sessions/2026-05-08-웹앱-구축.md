# 2026-05-08 웹앱 구축

## 작업 내용
iOS 앱과 동일한 기능의 React PWA 웹앱 신규 구축 및 배포 완료.

### 구현
- `web/` 디렉토리: React 18 + Vite + TypeScript + Tailwind CSS
- Google Identity Services (GIS) OAuth — 기존 iOS와 동일한 scopes
- GAS Script Execution API 호출 (`script.googleapis.com/v1/scripts/:run`)
- 4탭 구현: 대시보드 / 종목 / 분석 / 참고지표
- PWA manifest (홈화면 설치 가능)
- GitHub Actions (`deploy-web.yml`) → `gh-pages` 브랜치 자동 배포

### 배포
- URL: https://ejunwon-lab.github.io/FD5to6/
- Repo를 Public으로 변경 (보안 문제 없음 확인)
- GitHub Secret: `VITE_GOOGLE_CLIENT_ID` 등록
