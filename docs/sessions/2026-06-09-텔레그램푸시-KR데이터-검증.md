## 작업 방식 교정 (이번 세션에서 합의)
- 과거 실수: GitHub에서 도는 fetch를 **러너가 아닌 환경(로컬·US IP 프로브)에서 테스트**하고 "검증 완료"라 함. 환경이 다르면 검증이 아님.
- 앞으로: **실행 환경에서 진단 먼저 → 숫자 게이트로 판정 → 통과한 측정값만 "완료"**. 모든 완료 보고에 실환경 측정값 첨부.
