### 2. 종목 상세 버튼 위치 변경
- 웹앱·iOS 모두: 종목 카드 우측 상단/중앙 → **보유기간 라인 끝 인라인**
- HoldingCard에 `onDetail` 콜백 추가, 외부 overlay 제거

### 3. 전일 수익 = 전전일 데이터 표시 버그
- 원인: `prevDayChang*`을 *추이 기록* AJ2/AK2 백업 셀에서 읽었는데, 백업은 "거래일 첫 업데이트 시점"에만 갱신되어 항상 한 단계 늦음
- 해결: `_mFindPrevDayProfitChange()` — *추이 기록* U열에서 "어제 거래일" 행을 직접 검색해 AE/AF 반환. AJ/AK 백업 코드 제거

### 4. 분석 "오늘" 탭 자정 직후 0 문제 → priceAsOfDate 기반 라벨 통합
- 원인: 자정~09시엔 *현재가_이력*에 오늘 행 없음 → `_mCalcExtras`에서 todayIdx=prevIdx 같은 행 → change=0
- 해결:
  - `_mCalcExtras` fallback 강화: todayIdx=-1이면 마지막 행을 today로, 그 직전 거래일을 prev로
  - `newMobileGetPortfolio.summary`에 `priceAsOfDate` (*현재가_이력* 마지막 행 날짜) 추가
  - 클라이언트 `decideChangeLabel(priceAsOfDate)` 헬퍼 (Web `utils/changeLabel.ts`, iOS `Shared/ChangeLabel.swift`)
- 라벨 규칙:
  - priceAsOfDate === today → "오늘"
  - today 비거래일 → "최근"
  - priceAsOfDate === today-1 (달력상 어제) → "전일"
  - 그 외 → "최근"
- 적용: 분석 탭 / 종목 카드 "오늘 등락" / 대시보드 "오늘 수익" 카드 3곳 통일
