# 2026-05-09 웹앱 로그인 버그 수정

## 작업 내용

### 1. 실행@ — git pull 충돌 해결
- `deploy-web.yml` 로컬/원격 동시 수정으로 충돌 발생
- stash → pull → pop → 로컬 버전으로 해결 (VITE_GOOGLE_CLIENT_ID 환경변수)

### 2. Google 로그인 버튼 버그 수정
- **증상**: 웹앱에서 "Google로 로그인" 버튼 눌러도 아무 일도 안 일어남
- **원인**: `AuthContext.tsx`의 `signIn()` 함수가 `prompt: ''`(무음 모드)로 호출
  - 무음 모드는 기존 세션이 있을 때만 작동 → 처음 로그인 시 팝업 없이 조용히 실패
  - 버그 유입 경위: 앱 키 방식 시도 실패 후 OAuth로 복구하는 과정에서 잘못 남겨짐
- **수정**: `prompt: ''` → `prompt: 'select_account'`
- **파일**: `web/src/auth/AuthContext.tsx` L115

### 3. deploy-web.yml 환경변수 값 누락 수정
- 원격에 `VITE_GOOGLE_CLIENT_ID` (값 없음) 상태로 GitHub Actions 0초 실패 중이었음
- `VITE_GOOGLE_CLIENT_ID: ${{ secrets.VITE_GOOGLE_CLIENT_ID }}`로 수정
- gh CLI에 workflow 스코프 추가 후 push (`gh auth refresh -s workflow`)

### 4. 배포 완료
- GitHub Actions 18초 만에 성공
