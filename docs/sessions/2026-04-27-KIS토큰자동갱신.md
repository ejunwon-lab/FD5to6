# 2026-04-27 KIS 토큰 자동 갱신 트리거 추가

## 작업 내용

### 버그 진단: 모바일 앱 갱신 실패 (토큰 만료)

**증상**: 모바일 앱에서 업데이트가 안 됨. 시트에서 직접 "가격 갱신" 실행 후에는 모바일이 정상 동작.

**원인 파악**:
- KIS Access Token은 24시간 유효. Script Properties에 저장됨
- 토큰 만료 시 `KIS_API.getAccessToken()`이 재발급 시도
- 시트 직접 실행 컨텍스트에서는 재발급 성공 → Properties에 저장
- 모바일(Apps Script API) 컨텍스트에서는 재발급이 간헐적으로 실패
- 이후 모바일 재시도 시 이미 유효한 토큰을 사용해 성공 → "시트 갱신 후 모바일 작동" 패턴 확인

**근본 원인**: 토큰이 만료된 상태에서 모바일이 처음 호출할 때 재발급 실패. KIS API 간헐적 실패 또는 Apps Script API 실행 컨텍스트 차이로 추정.

### 해결: 매일 오전 8:30 자동 실행 트리거

`Main.js`에 트리거 등록/삭제 함수 추가:
- `setupDailyTrigger()`: 기존 트리거 삭제 후 매일 08:30 (Asia/Seoul) `runFullUpdate` 트리거 등록
- `deleteDailyTrigger()`: 트리거 삭제

메뉴 추가: `🛠️ 시스템 관리` → `⏰ 매일 8:30 자동실행 등록` / `🗑️ 자동실행 트리거 삭제`

**효과**: 매일 장 시작 전 `runFullUpdate`가 자동으로 돌면서 KIS 토큰 사전 갱신 + 전체 데이터 최신화. 이후 모바일 호출 시 항상 유효한 토큰 보장.
