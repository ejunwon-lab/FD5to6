## 완료 작업 (🔴 3개)

### #1 — 비거래일 buildDashboard 시 보유종목표 빈칸
- Dashboard.js `_calcExtraColumns`·`_calcTodayProfit`이 "오늘 행" 없으면 즉시 빈 결과
  → `updateNewPriceHistory` 거래일 가드 도입 후 주말마다 발동
- 해결: `_mCalcExtras`와 동일한 fallback 추가 (오늘 행 없으면 마지막 거래일 기준)
- errors.md 기록

### #2·#3 — 휴일 목록 (2026까지만 + 4곳 중복) → *휴장일* 시트 단일 소스
- 문제: 휴일 배열이 2026-12-31까지만(2027부터 오판) + GAS·web·iOS 4곳 중복.
  기존 목록은 이미 2026 추석도 누락 → 손 관리 실패 확인
- 해결:
  - **`Holidays.js` 신규** — `syncHolidays()`가 구글 '대한민국 공휴일' 캘린더 +
    KRX 고정휴장(5/1·12/31) → *휴장일* 시트. 음력·대체공휴일은 구글이 계산
  - `_isKoreanHoliday`가 *휴장일* 시트를 읽음 (시트 없으면 fallback)
  - GAS 중복 제거: MobileAPI `_isKoreanHoliday` 삭제, Trend `_trIsKoreanHoliday` 위임
  - 클라이언트(web·iOS) 휴일 배열 통째 삭제 → `decideChangeLabel`이 서버 `isTradingDay` 사용
  - 메뉴 `🗓️ 휴장일 동기화` + 매년 12/25 자동 트리거(`scheduledHolidaySync`)
