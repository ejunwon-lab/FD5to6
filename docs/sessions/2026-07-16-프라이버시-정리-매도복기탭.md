## 프라이버시 (핵심)
- **발견**: repo가 PUBLIC인데 docs·코드에 실제 원화 금액(순자산·잔액·손익·시드 실거래)이 커밋돼 공개돼 있었음.
- **B (기존 제거)**:
  - 1차 — docs·memory 전 히스토리의 집계 금액 리터럴 스크럽(git-filter-repo --replace-text, 코드 충돌 리터럴 제외). force push. 원격 실측 0건 확인.
  - 2차 — 죽은 함수 `importHistoricalTrades`(실거래 80건 하드코딩, 호출처 0) 삭제 + 히스토리 거래 행 리터럴 스크럽. force push. 원격 실측 0건.
  - 안전: 미러 백업 2개, 코드 무결성(node --check·vitest 49·tsc·build) 검증 후 push.
