# 2026-05-10 웹앱 탭 UI 개선

## 작업 내용

### 1. 분석 탭 전면 재구성 (AnalysisPage.tsx)
iOS AnalysisView.swift와 동일하게 맞춤.
- 투자 효율 매트릭스: 단기/장기 × 수익/손실 2×2 그리드, 탭하면 종목 목록 펼침
- 계좌별 분석: 현황 / 수익률 / 비중 / 오늘 / 종목 5탭
- 연 환산 수익률: 가로 바 차트, Y축 종목명 + 보유기간·수익률 2줄
- 52주 포지션: 위치 내림차순 정렬, 태그 (핵심 보유 / 잠자는 돈 / 고점 근접 / 깊은 손실 / 단기)
- 4개 섹션 아코디언 (하나만 열림)

### 2. 지표 탭 iOS 동일하게 재구성 (IndicatorsPage.tsx)
- 카테고리 순서 고정: 한국시장→한국선물→중국시장→미국시장→미국선물→AI/반도체→빅테크→상품→매크로
- 좌측 컬러바: 카테고리 색 → 등락 방향 색 (수익=빨강, 손실=파랑, 보합=회색)
- 행 레이아웃: 이름 / 현재값(88px) / 등락+등락률(72px VStack)
- 섹션 헤더: 카드 바깥 텍스트
- 숫자 포맷: ≥100이면 소수 2자리, <100이면 소수 3자리
- 갱신시각: 하단 "마지막 갱신"으로
- 새로고침 버튼: 유리 재질 원형

### 3. 탭 전환 시 재조회 제거 (App.tsx)
- 기존: 탭 전환마다 컴포넌트 언마운트→리마운트 → API 재호출
- 변경: 첫 방문 시만 로드, 이후 display:none으로 숨김 유지
- `visited: Set<Tab>` 상태로 방문 여부 추적
- 수동 새로고침 버튼은 기존대로 동작
