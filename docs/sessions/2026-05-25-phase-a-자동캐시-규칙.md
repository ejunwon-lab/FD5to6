## 주요 결정

- **숫자 절대 축약 금지** (`compact()`·`shortKRW` 등 헬퍼 제거) — `toLocaleString()` 풀 숫자만. 사용자 명시 규칙. → memory: `feedback_number_display`
- **종목명이 메인 표시, 종목코드는 보조** — 모든 카드/행/표/티커에 적용. → memory: `feedback_stock_name_primary`
- **GAS 트리거 시각 정확도**: `everyMinutes(30)` + 핸들러에서 분 체크 (슬랙 ±2분). `nearMinute`은 ±15분 슬랙이라 부적합
- **클라이언트 단일 데이터 소스**: 페이지마다 `usePortfolio` 새 instance → fetch 중복. Context로 끌어올려 Provider 1 instance가 모든 페이지 공유. 기존 hook 파일은 re-export 1줄로 변경해 호출처 import 무변경
- **시간당 자동 백그라운드 재페치**: GAS 09:30~16:30 매시 :30 자동 갱신과 매칭. 데스크 열어둔 채로도 항상 ≤1h fresh
- **ExposureMatrix 의도 재정의**: 시장 분류(KR/US) → 계좌별 손익(원금/수익/합계). 사용자 의도와 매트릭스 컬럼이 안 맞아 의미 자체를 변경

```
GAS                                  Web Desk
─────────────────────────────────    ──────────────────────────────────
09:30 매시 :30 (거래일만)             첫 진입
  → updateAllNew                        → DataProvider mount
  → KIS·환율·보유현황·추이·대시보드      → portfolio + indicators + history (Promise.all 3건)
  → 시트 캐시 갱신                       → 화면 표시
                                       → monthlyRealized prefetch (background)
                                        
                                     페이지 전환 (Holdings/Analysis/Indicators/Activity)

                                       → portfolio + realized 새로 받음
```
