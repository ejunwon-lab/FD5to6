데이터 시트 (NS 상수)

*거래_원장*
날짜·구분·종목코드·종목명·분류·증권사·계좌·수량·단가·금액·수수료·메모 (12컬럼). 매수/매도만 누적
*거래_입력폼*
사용자 거래 추가용 UI. 체크박스 submit → onEdit → 원장 추가
*현재가_이력*
행=날짜, 열=종목코드. KIS 가격 + 일별 누적. 1M/3M/6M/1Y 계산 소스
*보유현황*
종목코드·종목명·분류·증권사·계좌·보유기간·수량·평균단가·매입금액·현재단가·평가금액·손익·수익률·수동평가금액·비고 (15컬럼). 펀드/예금/보험은 K~M 셀에 수식 보존
*실현손익*
매도 시점·종목·수량·매도단가·매도금액·평균단가·매수원가·수수료·실현손익·수익률 (14컬럼)
*설정*
B2=USD/KRW, B3=GBP/KRW (GOOGLEFINANCE), 5행~ 대기 자금 항목 + 합계 행 (자동 감지)
*추이 기록*
(A) B~L 업데이트별 추이, (B) N~S 일별, (C) U~AF 수익. row 2=최신 스냅샷, AH/AI 캐시, AJ/AK 전일 백업
*대시보드*
20컬럼. [1]타이틀 [2]요약(4×2카드) [3]보유 종목 현황(정렬+합계) [4]계좌별 [5]분류별 [6]확정수익+Top/Bottom 5
참고지표 / _히스토리
KOSPI·KOSDAQ·KOSPI200·SPX·NDX·DJI·SOX·ES·NQ·GC·CL·VIX·TNX·DXY·NVDA·AAPL·MSFT·GOOGL·META·AMZN·TSLA·HSI (22개)
